Managing accounts

Administrator accounts can add and delete users, lock and unlock user accounts, and reset user passwords.

Adding a user

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Accounts heading, click the Add User button.

  4. Use the Add User dialog to create a new user:

    1. In the Email box, type the user's email address.

    2. Choose a setting from the Role drop-down list to indicate the type of user account (administrator or user) you want to create.

    3. Click Create User.

      A temporary password is generated and displayed in the confirmation dialog.

      Provide this password to new users and instruct them to change their passwords after logging in for the first time. For more information, see "Changing your password."

Deleting a user

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Accounts heading, click the Delete User button.

  4. Use the Delete User dialog to change your password:

    1. Choose a user from the User drop-down list.

    2. Click Delete User.

      Confirm you want to delete the selected user account, and then click Delete User again.

Locking a user account

Locking a user account allows you to temporarily suspend a user's account without removing the user from the Discussion Capture system. Locked user accounts are not permitted to sign in to the system and will be prompted to contact an administrator to unlock their accounts.

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Accounts heading, click the Lock User button.

  4. Use the Lock User dialog to choose an account:

    1. Choose a user from the User drop-down list.

    2. Click Lock User.

Unlocking a user account

Unlocking a user account resets a lock and allows a user to sign in again.

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Accounts heading, click the Lock User button.

  4. Use the Unlock User dialog to choose an account:

    1. Choose a user from the User drop-down list.

    2. Click Unlock User.

Resetting a user's password

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Accounts heading, click the Reset User Password button.

  4. Use the Reset User Password dialog to choose an account:

    1. Choose a user from the User drop-down list.

    2. Click Reset Password.

      A temporary password is generated and displayed in the confirmation dialog.

      Provide this password to the user and instruct him or her to change the password after logging in. For more information, see "Changing your password."
