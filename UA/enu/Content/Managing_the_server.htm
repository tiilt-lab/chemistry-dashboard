Managing the server

Administrator accounts can download and clear server and device logs to help analyze and troubleshoot the system.

Downloading server logs

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Server heading, click the Download Server Logs button.

  4. Use the Download Server Logs dialog to choose what you want to download:

    1. Choose a setting from the Service drop-down list to choose whether you want to download logs from the Discussion Capture Server or the Audio Processing Service.

    2. Click Download Logs.

      The selected log is downloaded to your Downloads folder as a .txt file.

Downloading device logs

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Server heading, click the Download Device Logs button.

  4. Use the Download Device Logs dialog to choose what you want to download:

    1. Choose a setting from the Device drop-down list to choose which pod or BYOD unit you want to review.

    2. Click Download Logs.

      The selected log is downloaded to your Downloads folder as a .txt file.

Clearing server logs

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Server heading, click the Clear Server Logs button.

  4. Use the Clear Server Logs dialog to choose which logs you want to clear:

    1. Choose a setting from the Service drop-down list to choose whether you want to clear logs from the Discussion Capture Server or the Audio Processing Service.

    2. Click Clear Logs.

      The selected log is reset.

Clearing device logs

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Settings button on the main Dashboard page:

  3. Under the Manage Server heading, click the Clear Device Logs button.

  4. Use the Clear Device Logs dialog to choose which logs you want to clear:

    1. Choose a setting from the Device drop-down list to choose which pod or BYOD unit log you want to clear.

    2. Click Clear Logs.

      The selected log is reset.
