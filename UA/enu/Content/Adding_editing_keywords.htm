Adding or editing keyword lists

Before you start a discussion, you need to create a keyword list. The keyword list defines terms that you want students to use in their discussion.

Creating a new keyword list

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Keyword Lists button on the main Dashboard page:

  3. On the Manage Keyword Lists page, click the New Keyword List button to display a new, untitled keyword list.

  4. Type a title for the new list, and then type a keyword or phrase in the box. Click Add Keyword or press Enter to add another keyword.

    Click to remove a keyword or phrase from the list.

  5. After you've entered all keywords, click Save to save the list.

Editing a keyword list

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Keyword Lists button on the main Dashboard page:

  3. On the Manage Keyword Lists page, click a keyword list to edit it.

  4. Click in the edit boxes to edit the list's title or keywords.

    Click to remove a keyword or phrase from the list.

  5. After you've edited the keyword list, click Save to save it.

Deleting a keyword list

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Keyword Lists button on the main Dashboard page:

  3. On the Manage Keyword Lists page, click on a keyword list and choose Delete from the menu.

  4. Click Delete in the confirmation dialog to remove the selected keyword list.
