Adding or disconnecting pods in the Dashboard

Your pods are preconfigured to work with Discussion Capture. However, the Dashboard allows you to add pods using a MAC address.

If you encounter difficulties with the Dashboard method, your IT staff can also use the Terminal to troubleshoot connection issues. For more information, see "Connecting pods to the network using Terminal."

Adding pods

  1. Find the pod's MAC address:

    1. Using the supplied USB cable, connect the microUSB port at the top of the pod to a USB port on your macOS or Linux computer.

    2. Open a Terminal window on macOS or Linux.

    3. Type the following command to display the path to the connected pod:

      macOS: ls /dev/cu.usb*

      Linux: ls /dev/ttyACM*

    4. Type the following command to establish a terminal session with the pod:

      macOS: screen /dev/cu.usbModem1141 115200

      Linux: screen /dev/ttyACM0 115200

      Please note that the specific device name will vary as in the following example from macOS:

    5. Sign in to the pod using the device's username and password.

    6. In Terminal, type ifconfig. The pod's MAC address is displayed.

      Use the eth0 MAC address for any network connections.

  2. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  3. Click the Pods button on the main Dashboard page:

  4. On the Manage Pods page, click Add via MAC address. A dialog box is displayed.

  5. Type the pod's MAC address that you recorded, and then click Add.

    The Discussion Capture server will detect the pod.

    The lights on the top of the pod will indicate its connection status:

    • While the pod attempts to connect to the dashboard, the lights will display a red, orange, yellow, green, blue, violet pattern.

    • When the pod has connected, the lights will pulse red.

    • If the pod cannot connect to the internet, a single red light is displayed after the red, orange, yellow, green, blue, violet pattern.

    • If the pod is not recognized, two red lights are displayed after the red, orange, yellow, green, blue, violet pattern. Please confirm the MAC address and repeat steps 4 and 5 above.

    • If the pod cannot communicate with the server, three red lights are displayed after the red, orange, yellow, green, blue, violet pattern. The dashboard server may be blocked or unreachable.

Disconnect a pod

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Pods button on the main Dashboard page:

  3. On the Manage Pods page, click the button on a pod and then click Remove.

  4. Click Remove in the confirmation dialog to disconnect the pod from the server.
