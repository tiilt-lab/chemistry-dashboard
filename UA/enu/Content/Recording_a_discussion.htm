Starting a discussion

After you've set up your pods and created a keyword list, you're ready to record a discussion.

  1. Using your computer or tablet, navigate to the Discussion Capture server and sign in.

  2. Click the Discussions button on the main Dashboard page:

  3. On the Manage Discussions page, click the New Discussion button.

  4. Use the Settings page to choose settings for your discussion:

    1. In the Discussion name box, type a name to identify your discussion.

    2. Click the button to choose the folder where you want to save the discussion.

    3. Select Allow participant devices if you want to allow students to join the discussion using their own devices. For more information, see "Joining a discussion with a student device."

    4. Select Capture discussion direction if you want the pod to capture information about the direction from which discussion was received. For more information, see "Reviewing keywords and questions."

    5. Select Analyze discussion features if you want to analyze qualities of the discussion. For more information, see "Joining a discussion with a student device."

    6. Click the Next button.

  5. Click the keyword list you want to use, or click Next to start recording without a keyword list.

  6. Select each pod that you want to record, and then click Start Discussion.

    If you want to record from a single student device, please see."Joining a discussion with a student device."

  7. Click a pod to view the discussion analytics for that device. For more information about analytics, please see "Reviewing keywords and questions" and "Reviewing discussion features."

    If students need assistance during a session, they can press the button on the top of the pod. After the button is pressed, the pod's icon will flash on the Overview page. Press the button again to turn off the call for help.

  8. When you're finished recording, click End.
